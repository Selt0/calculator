# JS Calculator

An on-screen calculator using JavaScript, HTML, and CSS.

## [live demo](https://selt0.github.io/calculator/)

---

### _Now featuring keyboard support!_

keyboard:

- **+ . - / %** : operators
- **0-9** : numbers
- **n** : +/-
- **c** : clear display
- **C** : hard clear
- **Backspace** : backspace
- **Enter** : equal
